# Colick Marketplace

Colick 的 Claude Code 插件市场。

## 包含的插件

| 插件 | 说明 |
|------|------|
| **colick-tools** | 并行子代理派遣与诚实执行规范 |

## 安装

```bash
/plugin marketplace add Pulck/ColickTools
/plugin install colick-tools@colick-marketplace
```

## 更新

```bash
/plugin update colick-tools@colick-marketplace
```

或更新 marketplace 下所有插件：

```bash
/plugin update --all
```
