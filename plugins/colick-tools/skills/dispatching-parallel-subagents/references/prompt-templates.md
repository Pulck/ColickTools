# Prompt 模板

## 判定型模板

```
检查 [文件路径] 中是否包含 [关键词/模式]。

判定标准：[明确的通过/失败条件]

返回格式：是 / 否（一句话，无解释）
```

## 实现型模板

```
实现模块：[模块名称]

接口契约：
- 输入：[参数及类型]
- 返回：[返回值及类型]
- 错误：[错误处理要求]

验收标准：
1. [可自动验证的条件 1]
2. [可自动验证的条件 2]

约束：
- [技术栈限制]
- [性能要求]
- [代码规范]

返回格式：
- 状态：DONE / ERROR / BLOCKED
- 产物：[代码块或文件路径]
- 自测结果：[简述测试覆盖和通过情况]
```

## 二级子代理

实现型子代理内部若遇到独立的判定型任务（如"检查依赖版本是否满足"），可自行派遣二级子代理。**二级子代理应只做判定型任务，不应承担核心模块实现工作。**
